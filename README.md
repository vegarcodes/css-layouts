# CSS-layouts på 1-2-3!

Denne CSS-fila er resultatet av en liten talk jeg holdt på [Norsk programmering](https://norskprogrammering.no) om CSS-layouts og hvordan vi kan gjøre det litt enklere for oss selv.

Tanken er at dette skal være et utgangspunkt for videre tukling og utvikling, ut fra hva du faktisk trenger i prosjektet ditt. Meget opinionated. Bruk om det gir deg glede og nytte.
